# CreditCardAPI
API REST de pagamentos de cartão de crédito

**Pré requisitos:** Docker e Docker Compose

```
# Criar e levantar os containers
docker-compose up -d

# Iniciar o container Docker
docker-compose start

# Parar o container Docker
docker-compose stop

```

**Acesso localhost**

```
http://localhost:8000

```

**Acesso phpMyAdmin**

```
http://localhost:8080

servidor: mysql
user: root
senha: root

```
